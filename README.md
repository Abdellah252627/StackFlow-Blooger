# 🚀 StackFlow Blogger

مدونة تقنية عصرية مبنية بأحدث تقنيات الويب، تجمع بين التصميم الفاخر (Glassmorphism) والأداء العالي.

## ✨ المميزات الرئيسية
- **🎨 تصميم عصري:** واجهة داكنة مذهلة مع تدرجات لونية متحركة.
- **🔐 نظام الدخول:** دعم كامل لتسجيل الدخول عبر البريد الإلكتروني وحسابات جوجل (OAuth).
- **✍️ منشئ المحتوى:** محرر نصوص غني (Rich Text Editor) يدعم تنسيق النصوص والصور.
- **⚙️ لوحة تحكم الإدارة:** إمكانية تغيير عنوان المدونة، وصفها، واختيار القالب البصري.
- **🛡️ طبقات أمان:** حماية من هجمات XSS، Spam (Honeypot)، وتشفير كلمات المرور عبر Firebase.
- **📱 متوافق مع كل الشاشات:** تصميم متجاوب (Responsive) يعمل ببراعة على الجوال والحاسوب.

## 🛠️ التقنيات المستخدمة
- **Frontend:** HTML5, Vanilla CSS, JS (ES6+).
- **Bundler:** Vite.
- **Backend/Auth:** Firebase Authentication.
- **Components:** Quill.js, Font Awesome, Google Fonts.

## 🚀 طريقة التشغيل
1. قم بتحميل المشروع (Clone).
2. افتح المجلد في Terminal.
3. قم بتثبيت المكتبات: `npm install`.
4. قم بتهيئة ملف `js/firebase-config.js` ببيانات مشروعك في Firebase.
5. شغّل المشروع محلياً: `npm run dev`.

## 📂 هيكل الملفات
- `/css`: ملفات التنسيق.
- `/js`: المنطق البرمجي وإعدادات Firebase.
- `/pages`: الصفحات الفرعية (لوحة التحكم والمحرر).
- `index.html`: الصفحة الرئيسية.
